Timestamp Microservice

This microservice will return a JSON object contianing both the unix timestamp and the natural language date string when provided with either counterpart. It was built by me, rob j loranger as part of the back end developer program at FreeCodeCamp.

Example Query

http://robsfcctimestamps.herokuapp.com/api?time=July%201,%201988
http://robsfcctimestamps.herokuapp.com/api?time=583743600

Example Returned JSON

{"unix":583743600, "natural":"July 1, 1988"}
